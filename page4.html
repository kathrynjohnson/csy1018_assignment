Resource Page

This page features all the best websites that are useful for people who want to learn about web development, HTML and CSS.

W3Schools
W3Schools is a basic websites that teaches individuals about HTML, CSS, Javascript and other web based tutorials. The website is good for giving examples.
Codecademy
Codecademy is a website that is free to learn about coding. There are plenty of corses that you can take. They have a large range of courses from HTML to CSS.
edX
EdX is a website that was created by Harvard University and MIT back in 2012.
CodePen.io
CodePen is a website where individuals can shared ideas that they have done. Individuals can look at other individuals work and take inspiration from what they find.
Khan Academy
Khan Academy is a website created in 2006 by Salman Khan. This site has step-by-step videos on courses such as HTML, CSS and JavaScript.
HTML5 Rocks
HTML5 is a website created by Google that was created in 2010. This website has loads of resources and HTML5 updates that individuals can use to increase their knowledge
Lynda.com
Lynda.com is a very good website that gives online tutorials for all kinds of technology based courses. This does inlude helpful tutorials on HTML and CSS.
YouTube
People may not realise that there people put hundreds of videos on YouTube to help with web developmemt. It really is surprising what you can find on YouTube.
CSSDeck
CSSDeck is a website that people can use to help them solve any coding problems that they have.
