Glossary

There are terms that are used in Web Development

A
Alt Tags - Text that descibes the image
C
CSS - the cascading style sheet that developers use to design their html pages
D
Domain - the name of the website that individuals will type into the browser to locate it
F
Flexbox
Float - this is positioning on the page
H
HTML - this stands for Hypertext Markup Language which is the markup language that developers use on their websites
I
Images - images are very common on webpages and it is also important that you optimise them
M
Media Queries
N
Navigation - this is alos known as the 'Menu Bar', it is usually located at the top of the screen and it is sued to go between different pages on a website
R
Root Directory
S
Slack - Slack is a website that allows individuals to talk to others and create channels. A lot of major companies get their employees on Slack
T
Thimble [moz:lla]- is an online code editor
V
Validator - is a site that checks if you've got any errors in your code
W
Wireframes - wireframes are the first sketch of what a website will look like before any designs have been added to it
#
404 - this os what happens when a webpage is non-existent
